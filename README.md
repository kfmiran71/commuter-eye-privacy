# Commuter Eye Website

Static, mobile-first public website for Commuter Eye.

## Preview locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Contents

- `index.html` - landing page, beta CTA, features, support/contact
- `privacy-policy/index.html` - public privacy policy page
- `styles.css` - shared site styling
- `assets/` - public logo and favicon only

## Publishing

This folder can be published through a static host such as Vercel, Netlify, GitHub Pages, or uploaded/rebuilt in Wix. If hosting outside Wix, point `commutereye.com` DNS records from Wix to the selected host.
